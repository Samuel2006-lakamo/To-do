📝 Todo List App

A simple, performance-focused todo list application built with vanilla HTML, CSS, and JavaScript to help you manage your tasks efficiently.

✨ Features

  • Lightweight & Fast - Built with vanilla technologies for optimal performance
  • Simple Interface - Clean, intuitive design for easy task management
  • Responsive Design - Works seamlessly across all devices
  • No Dependencies - Pure HTML, CSS, and JavaScript implementation

🚀 Quick Start

Option 1: Use the Live App

Ready to start organizing your tasks? 🌐 Open Todo List App

Option 2: Run Locally

Prerequisites

  • Git installed on your system
  • A modern web browser

Installation

  1. Clone the repository
git clone https://github.com/Peakk2011/TodoList.git
  1. Navigate to the project directory
cd TodoList
  1. Open in your browser
# Simply open index.html in your preferred browser
   open index.html
